# 📊 Grafana GIFs Repository

Este repositório é utilizado para armazenar **GIFs** que são usados nos **dashboards do Grafana** 🎯.  
A ideia é manter todos os GIFs organizados e versionados para facilitar a personalização visual dos painéis.

## 🚀 Como Usar no Grafana

1. Faça upload dos GIFs no servidor onde o Grafana está rodando.
2. No painel do Grafana, adicione um **Text Panel** ou **Image Panel**.
3. Insira o caminho ou URL do GIF. Exemplo:

```

![Server Status](/public/gifs/server-status.gif)

```

4. Ajuste o tamanho e layout conforme necessário no painel.

## 📦 Regras para Contribuição

- Somente GIFs otimizados e leves (preferencialmente abaixo de 2MB).
- Nomeie os arquivos de forma descritiva e use `-` para separar palavras (exemplo: `server-down.gif`).
- Mantenha a raiz do repositório limpa, sem subpastas.

## 💡 Dica

Você pode usar ferramentas como [ezgif.com](https://ezgif.com/) para redimensionar e otimizar seus GIFs antes de adicioná-los aqui!
